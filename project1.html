Monster Hunt

Monster Hunt is basic game project created with Html Csss and JavaScript . It is a game where a user can click good object and bad object where, clicking a good object gives you a point and clicking a bad object will reduce your lives. This game has a time limit, when the time limit is over the game will end automatically.

I designed and created graphics, logo and illustrations for this project from Photoshop and Illustraor. This is a mandatory school project from first semester, where i was introduced to basic animation and basic JavaScript.
